## Task API - Flask

This project is a simple REST API built with Flask to manage tasks. It implements the basic CRUD operations, allowing you to create, read, update, and delete tasks.

The API uses an in-memory list to store data. making it ideal for learning and practicing backend concepts such as routing, HTTP methods, and JSON handling.

### Features
- Create new tasks  
- List all tasks  
- Get a task by ID  
- Update a task  
- Delete a task  

### Technologies
- Python  
- Flask  

### Purpose
This project was developed to practice backend development, REST API structure, and improve technical English in code documentation.

## API de Tarefas - Flask

Este projeto é uma API REST simples construída com Flask para gerenciar tarefas. Ele implementa as operações básicas de CRUD, permitindo criar, ler, atualizar e deletar tarefas.

A API utiliza uma lista em memória para armazenar os dados, justamente para aprendizado e prática de conceitos de backend como rotas, métodos HTTP e manipulação de JSON.

### Funcionalidades
- Criar novas tarefas  
- Listar todas as tarefas  
- Buscar uma tarefa por ID  
- Atualizar uma tarefa  
- Deletar uma tarefa  

### Tecnologias
- Python  
- Flask  

### Objetivo
Este projeto foi desenvolvido para praticar desenvolvimento backend, estrutura de APIs REST e melhorar o inglês técnico na documentação de código.
